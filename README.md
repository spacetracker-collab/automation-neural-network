# Automation Neural Network (AutoNN)

This repository implements the **Automation Neural Network (AutoNN)** model described in the paper:

Enterprise Automation as a Neural Network

## Concept

Enterprise RPA ecosystems can be represented as neural networks:

bots → neurons  
workflows → synapses  
tasks → signals  

The AutoNN model treats automation workflows as graph neural networks.

## Features

- Graph-based automation model
- Softmax task routing
- Gradient-based optimization
- Enterprise workflow simulation
